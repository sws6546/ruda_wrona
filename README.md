# Ruda Wrona Or≈Ça Nie Pokona (school project)

download:
```
git clone https://github.com/sws6546/ruda_wrona
cd ruda_wrona
yarn install
```

config:
* add `.env.local` in root of directory
* put env variables:
```
MONGODB_URI=<url for mongodb>
JWT_TOKEN=<eg. secret>
```

and just run `docker compose up`
